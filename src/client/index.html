What is RealQuest?

RealQuest is a monster-hunter game in which the player has to shoot enemies in a realistic environment, generated using the mapbox API. Additional APIs are used for day/night cycles and weather reporting. The player navigates the environment while avoiding hordes of monsters and gaining as many points as it can.
